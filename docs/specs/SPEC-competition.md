## 一、模块定位

赛事模块面向专业青少年选手、家长、教练和乒乓球爱好者，提供三类核心能力：

1. **赛事日历**：只展示正规、权威、专业的大赛信息，帮助用户快速掌握当前月份的重要赛程。
2. **青少年赛事**：区分为“专业赛事”和“业余赛事”，让用户明确区分官方正规赛事与民间积分赛事。
3. **技战术分析**：以高质量文字解析为主，辅以视频链接，沉淀经典大赛的技战术复盘内容。

## 二、赛事数据来源与分类原则

### 2.1 数据来源

| 来源 | 方式 | 覆盖 | 状态 |
|------|------|------|------|
| 中国乒协官网 (ctta.cn) | 直接 HTTP 爬虫 | 官方公告、全国赛事、青少年正规赛事 | ✅ 已实现 |
| 搜狗微信搜索 (weixin.sogou.com) | HTTP 爬虫 + 关键词轮询 | 公众号赛事通知、青少年赛事、业余积分赛 | ✅ 已实现 |
| 搜狗视频 (v.sogou.com) | HTTP 爬虫 + 聚合结果 | B站 / 腾讯视频 / 爱奇艺 / 抖音等战术视频 | ✅ 已实现 |
| 手动录入 / seed | 种子脚本 + 管理 API | 历史经典赛事、精选专题 | ✅ 已部署 |

### 2.2 分类规则

统一在 MongoDB `events` 集合中新增字段：`competitionLevel`

| 值 | 含义 |
|----|------|
| `professional` | 专业正规赛事：WTT、世乒赛、世界杯、全国锦标赛、全运会、省运会、乒超、八大杯、全国青少年正规赛事等 |
| `amateur` | 业余赛事：爱聚网、开球网、民间积分赛、地方业余赛等 |
| `unknown` | 无法可靠分类的内容 |

### 2.3 当前推荐的展示原则

- **赛事日历**：只显示 `competitionLevel=professional`，重点展示国际/国内正规大赛。
- **青少年赛事**：
  - **专业赛事**：全国少年/少儿/青年锦标赛、全运会青少年组、省运会青少年组、八大杯、俱乐部联赛、全国青少年选拔/集训等。
  - **业余赛事**：爱聚网、开球网、地方积分赛、民间交流赛等。
- **技战术分析**：以“经典比赛复盘文字内容”为主，视频仅作为补充链接。

## 三、赛事日历设计

### 3.1 定位

赛事日历只展示正规、权威、专业的比赛，避免混入业余赛、培训通知、行政通知等噪音内容。

### 3.2 展示内容

- WTT 大满贯 / 冠军赛 / 球星挑战赛
- 世界杯 / 世乒赛 / 亚锦赛 / 亚运会 / 奥运会
- 全国锦标赛 / 全国青年锦标赛 / 全国少儿锦标赛
- 全运会 / 省运会 / 乒超联赛 / 官方俱乐部赛

### 3.3 前端筛选

- 月份选择器
- 分类筛选：全部 / 国际 / 国内
- 状态标签：即将开始 / 进行中 / 已结束
- 仅展示 `competitionLevel=professional`

### 3.4 详情页要素

赛事卡片点击进入详情页后，建议至少展示：

- 赛事名称
- 日期区间
- 地点
- 主办方
- 组别
- 报名截止日期
- 报名方式 / 报名链接
- 竞赛规程正文（如果可抓到）
- 来源链接

## 四、青少年赛事设计

### 4.1 定位

青少年赛事需要明确区分**专业正规赛事**和**业余赛事**，避免用户被杂乱赛事干扰。

### 4.2 专业赛事

建议包含：

- 全国少年 / 少儿 / 青年锦标赛
- 全国青少年选拔赛 / 集训营 / 等级测试
- 全运会青少年组
- 省运会青少年组
- 八大杯赛（创新杯 / 新星杯 / 娃娃杯 / 向阳杯 / 开拓杯 / 奥星杯 / 四环杯 / 幼苗杯）
- 俱乐部联赛青少年组

### 4.3 业余赛事

建议包含：

- 爱聚网积分赛 / 联赛
- 开球网积分赛 / 万元赛
- 地方性民间邀请赛
- 各类社会俱乐部赛事

### 4.4 前端展示

青少年赛事 tab 建议分成两个一级入口：

1. **专业赛事**
   - 子分类：全国赛 / 八大杯 / 省赛 / 俱乐部 / 选拔集训
   - 搜索：赛事名 / 城市 / 主办方

2. **业余赛事**
   - 搜索：赛事名 / 城市 / 关键词
   - 重点突出“积分赛”“民间赛”“交流赛”

## 五、赛事详情页设计

### 5.1 核心原则

详情页要提供足够的信息量，满足用户“点进去就能看懂这是什么赛事、何时举行、怎么报名、在哪里举办”的需求。

### 5.2 建议结构

1. 基本信息
   - 赛事名称
   - 标签：专业 / 业余、国际 / 国内、进行中 / 已结束
   - 日期
   - 地点
   - 主办方

2. 报名信息
   - 报名截止时间
   - 报名方式
   - 报名链接

3. 竞赛规程
   - 正文内容
   - 比赛日程
   - 参赛要求
   - 注意事项

4. 来源信息
   - 原文链接
   - 来源平台

5. 关联内容
   - 相关技战术分析
   - 相关视频

## 六、技战术分析模块

### 6.1 定位

技战术分析模块以**高质量文字点评/复盘**为主体，视频作为辅助链接。

### 6.2 内容形态

| 形态 | videoUrl | articleContent | 场景 |
|------|:---:|:---:|------|
| 纯视频 | ✅ | 空 | B站、视频号、抖音等视频资源 |
| 纯文章 | 空 | ✅ | 公众号、专栏深度复盘 |
| 视频 + 文章 | ✅ | ✅ | 经典比赛完整复盘 |

### 6.3 前端展示

- 主题：经典比赛 / 技术专题
- 时间范围：近3月 / 近半年 / 近1年 / 全部
- 标签筛选：反手、正手、发球、接发球、步法、相持、关键分等
- 文章正文优先展示
- 视频作为“相关视频”挂载在详情页底部

### 6.4 详情页布局

#### 顶部
- 标题
- 对阵双方 / 技术主题
- 赛事名称
- 比赛日期
- 标签

#### 内容主体
- 比赛背景
- 技术要点
- 战术博弈
- 数据/节奏/落点分析
- 学习启示

#### 辅助内容
- 相关视频
- 原文链接
- 来源账号 / 来源平台

### 6.5 视频号 / B站 / 抖音推荐

建议统一用配置文件维护：

- 微信视频号：`wx.openChannelsUserProfile`
- B站：`wx.navigateToMiniProgram`
- 抖音：提示搜索 / 复制账号名
